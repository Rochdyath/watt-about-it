# Plan de réponse à incident

Détection
- Alerte CI ou alerte cloud

Confinement
- Révoquer clés et tokens
- Désactiver endpoints exposés

Éradication
- Rotation des secrets
- Purge des artefacts compromis

Rétablissement
- Déploiement contrôlé
- Tests de régression

Leçons
- Post-mortem et actions préventives
