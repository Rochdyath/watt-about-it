# Risk Register – Watt about it

## Objectif
Identifier et documenter les risques liés au traitement des données.

## Risques principaux
- Fuite de données à cause d’un bucket mal configuré
- Secrets exposés dans Git
- Logs trop bavards contenant potentiellement des informations sensibles
- Modèle entraîné sur un dataset incorrect

## Atténuation
- Buckets privés et IAM minimal
- .env hors Git + Secret Manager
- Validation automatique des logs
- Contrôle de schéma à l’ingestion
